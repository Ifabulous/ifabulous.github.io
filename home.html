ANALYSING COVID-19 TRENDS IN AFRICA AND EUROPE

I analysed the Our World in Data COVID-19 Dataset to identify key factors and identify the trends using Python. Visualization was done using Matplotlib and Seaborn Libraries. An Interactive element was also added to the code.

PREDICTING EMPLOYEE ATTRITION USING MACHINE LEARNING

Three machine learning algorithms were used to predict employee attrition, namely, decision trees, support vector machines (SVM) and K-Nearest neighbours (KNN). The dataset was sourced from Kaggle, consisting of 10 features and 14,249 rows consisting of a variety demographic, job-related, and performance-related variables.

CASTING STEEL DEFECT DETECTION

This Deep Learning Image Classification task seeks to investigate the application of convolution neural networks (CNN) using the ResNet 18 model in the detection of defects in the real life industry casting products dataset containing 7348 images of a submersible pump impeller sourced from Kaggle. The dataset was transformed, normalised and split into training, validation and test sets.

MURDER MYSTERY CASE

In this project, a hypothetical murder case was solved using SQL. A crime had taken place, and the detective needed help to solve the mysterious case. He gave you a crime report but you somehow lost it. You vaguely remember that the crime was a murder that occurred sometime on January 15, 2018. Will the murderer be found?

IMPLEMENTATION OF A PARKING RENTAL APP DATABASE

In this project, a scenario-based implementation of a relational database for a parking rental app was carried out after the conceptual and logical design phase
